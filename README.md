# Super-Simple-Discord-Bot

## What is this?
This is an extremely basic Discord bot with a few commands, a simple role system and a customizable role system.

## How to use it

**1)** First, you must create a discod bot using the developer portal ensuring to enable:

* Manage Roles
* View Channels
* Send Messages
* Create Public Threads
* Manage Messages
* Manage Threads
* Embed Links
* Attatch Files
* Read Message History
* Use external Emojis
* Add reactions
* Create Polls

(Not all of these are in use currently)

**2)** Next, add the bot to your server

**3)** After that, customize the chat filter and custom role

**4)** Finally, add your bot token into the .env file and you're good to go!
